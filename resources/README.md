# 图标资源说明

## 图标文件

将你的图标文件命名为 `app_icon.ico` 并放在此目录下。

## 图标要求

- **格式**: .ico 文件
- **推荐尺寸**: 包含多个尺寸（16x16, 32x32, 48x48, 256x256）
- **用途**: 系统托盘图标

## 如何创建图标

### 方法 1: 在线工具
1. 访问 https://www.icoconverter.com/ 或 https://convertio.co/zh/png-ico/
2. 上传你的 PNG/JPG 图片
3. 选择多个尺寸（16x16, 32x32, 48x48）
4. 下载生成的 .ico 文件
5. 重命名为 `app_icon.ico` 并放在此目录

### 方法 2: 使用 GIMP
1. 打开 GIMP
2. 导入你的图片
3. 调整大小为 256x256
4. 导出为 .ico 格式
5. 选择包含多个尺寸

### 方法 3: 使用 ImageMagick
```bash
magick convert input.png -define icon:auto-resize=256,128,64,48,32,16 app_icon.ico
```

## 图标设计建议

- 使用简单、清晰的设计（在小尺寸下也能识别）
- 考虑使用键盘或修饰键相关的图标
- 建议使用透明背景
- 颜色对比度要高，便于在不同背景下识别

## 示例图标主题

- 键盘图标
- Ctrl/Shift/Alt 按键图标
- 修复/工具图标
- 盾牌图标（表示保护）
- 齿轮图标（表示自动化）

## 当前状态

如果 `app_icon.ico` 不存在，程序将无法编译。请确保添加图标文件后再编译。
